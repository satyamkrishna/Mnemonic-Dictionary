      <h2>oracular</h2>
      <p><strong>Short Definition</strong> : of an oracle; prophetic; uttered as if with divine authority; mysterious or ambiguous; hard to understand; N. oracle: wiser person; prophecy made by an oracle</p>

         
      <div style="padding-bottom:0px;">
         

         <u>Definition</u><br />      
         (adj) of or relating to an oracle
         
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  able by oracular means to expose a witch               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (adj) obscurely prophetic
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/delphic">delphic</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  Delphic pronouncements               </li>
                              <li>
                  an oracular message               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (adj) resembling an oracle in obscurity of thought
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/enigmatic">enigmatic</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  the oracular sayings of Victorian poets               </li>
                              <li>
                  so enigmatic that priests might have to clarify it               </li>
                              <li>
                  an enigmatic smile               </li>
