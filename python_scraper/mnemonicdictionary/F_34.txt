      <h2>fussy</h2>
      <p><strong>Short Definition</strong> : fastidious; finicky; easily upset</p>

         
      <div style="padding-bottom:0px;">
         

         <u>Definition</u><br />      
         (adj) annoyed and irritable
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/bad-tempered">bad-tempered</a>

                        <a href="http://www.mnemonicdictionary.com/word/crabbed">crabbed</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/crabby">crabby</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/cross">cross</a>

                        <a href="http://www.mnemonicdictionary.com/word/grouchy">grouchy</a>

                        <a href="http://www.mnemonicdictionary.com/word/grumpy">grumpy</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/ill-tempered">ill-tempered</a>

            
             
         </div>
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (adj) overcrowded or cluttered with detail
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/busy">busy</a>

         <u>Definition</u><br />      
         (adj) exacting especially about details
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :
