      <h2>knit</h2>
      <p><strong>Short Definition</strong> : contract into wrinkles; grow together; join together closely; make (a fabric or garment) by interwining yarn or thread; Ex. knit the brow; Ex. The bones should knit together in a few weeks.</p>

         
      <div style="padding-bottom:0px;">
         

         <u>Definition</u><br />      
         (noun) a fabric made by knitting
         
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) a basic knitting stitch
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/knit stitch">knit stitch</a>

                        <a href="http://www.mnemonicdictionary.com/word/plain">plain</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/plain stitch">plain stitch</a>

            
             
         </div>
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) needlework created by interlacing yarn in a series of connected loops using straight eyeless needles or by machine
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/knitting">knitting</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/knitwork">knitwork</a>

            
             
         </div>
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (verb) make (textiles) by knitting
         
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>

                  knit a scarf               </li>

         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (verb) tie or link together
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/entwine">entwine</a>

            
             
         </div>
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>
