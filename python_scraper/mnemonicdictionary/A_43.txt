      <h2>application</h2>
      <p><strong>Short Definition</strong> : diligent attention; diligence; V. apply oneself</p>

         
      <div style="padding-bottom:0px;">
         

         <u>Definition</u><br />      
         (noun) the act of bringing something to bear; using it for a particular purpose
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/practical application">practical application</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  he advocated the application of statistics to the problem               </li>
                              <li>
                  a novel application of electronics to medical diagnosis               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) a verbal or written request for assistance or employment or admission to a school
         
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  December 31 is the deadline for applications               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) the work of applying something
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/coating">coating</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/covering">covering</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  the doctor prescribed a topical application of iodine               </li>
                              <li>
                  a complete bleach requires several applications               </li>
                              <li>
                  the surface was ready for a coating of paint               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) a program that gives a computer instructions that provide the user with tools to accomplish a task
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/application program">application program</a>

         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>

                  he has tried several different word processing applications               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) liquid preparation having a soothing or antiseptic or medicinal action when applied to the skin
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/lotion">lotion</a>

         <u>Definition</u><br />      
         (noun) a diligent effort
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/diligence">diligence</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  it is a job requiring serious application               </li>
                           </ul>
         </div>
