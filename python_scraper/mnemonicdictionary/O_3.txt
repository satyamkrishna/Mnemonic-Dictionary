      <h2>obliterate</h2>
      <p><strong>Short Definition</strong> : destroy completely; wipe out; Ex. obliterate the village</p>

         
      <div style="padding-bottom:0px;">
         

         <u>Definition</u><br />      
         (verb) mark for deletion, rub off, or erase
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/kill">kill</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/wipe out">wipe out</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  kill these lines in the President's speech               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (verb) make undecipherable or imperceptible by obscuring or concealing
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/blot out">blot out</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/hide">hide</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/obscure">obscure</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/veil">veil</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  a hidden message               </li>
                              <li>
                  a veiled threat               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (verb) remove completely from recognition or memory
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/efface">efface</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  efface the memory of the time in the camps               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (verb) do away with completely, without leaving a trace
         
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (adj) reduced to nothingness
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/blotted out">blotted out</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/obliterated">obliterated</a>

            
             
         </div>
