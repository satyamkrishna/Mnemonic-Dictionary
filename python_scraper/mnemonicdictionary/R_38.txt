      <h2>revert</h2>
      <p><strong>Short Definition</strong> : relapse; backslide; turn back to; return to the former owner; N. reversion</p>

         
      <div style="padding-bottom:0px;">
         

         <u>Definition</u><br />      
         (verb) go back to a previous state
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/regress">regress</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/retrovert">retrovert</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/return">return</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/turn back">turn back</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  We reverted to the old rules               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (verb) undergo reversion, as in a mutation
         
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

               </div>
