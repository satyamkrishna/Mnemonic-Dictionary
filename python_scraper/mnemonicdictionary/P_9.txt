      <h2>parity</h2>
      <p><strong>Short Definition</strong> : equality; close resemblance; CF. disparate</p>

         
      <div style="padding-bottom:0px;">
         

         <u>Definition</u><br />      
         (noun) (obstetrics) the number of liveborn children a woman has delivered
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/para">para</a>

            
             
         </div>
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  the parity of the mother must be considered               </li>
                              <li>
                  a bipara is a woman who has given birth to two children               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) (mathematics) a relation between a pair of integers: if both integers are odd or both are even they have the same parity; if one is odd and the other is even they have different parity
         
          

                 
         <div style="padding-top: 10px;"><u>Example Sentence</u> <br />
            <ul>
                              <li>
                  parity is often used to check the integrity of transmitted data               </li>
                           </ul>
         </div>
            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) (computer science) a bit that is used in an error detection procedure in which a 0 or 1 is added to each group of bits so that it will have either an odd number of 1's or an even number of 1's; e.g., if the parity is odd then any group of bits that arrives with an even number of 1's must contain an error
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/check bit">check bit</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/parity bit">parity bit</a>

            
             
         </div>
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) (physics) parity is conserved in a universe in which the laws of physics are the same in a right-handed system of coordinates as in a left-handed system
         
                  <div style="padding-top:8px;">
            <u>Synonyms</u> :           
                        <a href="http://www.mnemonicdictionary.com/word/conservation of parity">conservation of parity</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/mirror symmetry">mirror symmetry</a>

                        ,
            
                        <a href="http://www.mnemonicdictionary.com/word/space-reflection symmetry">space-reflection symmetry</a>

            
             
         </div>
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>

         

         <u>Definition</u><br />      
         (noun) functional equality
         
          

            
         <hr style="margin-top:24px;margin-bottom:5px;"/>
